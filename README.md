# Machine Learning Implementation Library 

## Overview
A comprehensive reference repository demonstrating the practical application of core Machine Learning algorithms. This collection includes from-scratch mathematical implementations as well as optimized models utilizing industry-standard libraries. 

This repository serves as a foundational toolkit for predictive modeling, exploratory data analysis (EDA), and algorithmic evaluation across both supervised and unsupervised learning domains.

## Contents & Algorithms
* **Supervised Learning:** 
  * Linear & Logistic Regression
  * Decision Trees & Random Forests
  * Support Vector Machines (SVM)
  * K-Nearest Neighbors (KNN)
* **Unsupervised Learning:**
  * K-Means Clustering
  * Principal Component Analysis (PCA) for dimensionality reduction
* **Model Evaluation:** Cross-validation, hyperparameter tuning, and metric analysis (Accuracy, Precision, Recall, F1-Score, RMSE).

## Tech Stack
* **Language:** Python
* **Data Manipulation:** NumPy, Pandas
* **Machine Learning:** Scikit-Learn
* **Data Visualization:** Matplotlib, Seaborn
